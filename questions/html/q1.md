Title: Create a Job Application Form
Description:

Build a form for job applications. The form should allow candidates to enter their details and upload their resume.


Task

Add an input field for Full Name


Add an input field for Email (should validate email format)


Add an input field for Phone


Add a File Upload option for the resume


Add a Submit button to complete the form



Hints

Use the <form> tag to group all fields.


Use <label> with each <input> for accessibility.


For email validation, use <input type="email">.


For file upload, use <input type="file">.


Add name attributes to inputs (useful for backend later).


Use <button type="submit"> for the final action.
